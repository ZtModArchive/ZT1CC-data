One of the most interesting and challenging features of Marine Mania is the ability to combine land exhibits with tank exhibits to create large, multiple environment exhibits for marine (and possibly other) species. This final tutorial will show you how to create these exquisite exhibits for your marine wildlife.

Before we continue, please note that if at any time you wish to exit the tutorial, you can click on the Game Options tab (blue disk) on the left side of the game screen and select Main Menu.

Click the OK button to start the tutorial.
